# MYCHP203-TOP-Project : Seismic core - 3D stencil optimization

Ce projet a été réalisé par Martin Diamant et Lisa Taldir.

## Utilisation

Pour exécuter `top-stencil`, se placer dans le répertoire courant puis faire :

```bash
$ mkdir build
$ cd build
$ cmake ..
$ make
$ cd ..
$ build/top-stencil
```
